|------|---------|

| **HCPR** | Helpful, Constructive, Positive, Real — thought-challenging toolkit |
| **DTR** | Daily Thought Record / “thought on trial” |
| **Core Skills** | Engine skills used with conceptualisation (distancing, thought tools) |

| **Phase 2** | Goal setting |
| **Phase 3** | Skill application |
